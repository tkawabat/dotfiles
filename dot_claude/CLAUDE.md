# CLAUDE.md (User-level / Home)

すべての Claude Code セッションで読み込まれる、ユーザー個人のグローバル設定。
プロジェクト固有の指示は各プロジェクト直下の `CLAUDE.md` に書く。

## 言語

- 日本語で応答する。

## 一般的なふるまい

- 不要な前置きや末尾の要約は省く。差分やコマンド出力で伝わることを文章で繰り返さない。
- 不確実な点を勝手に推測で埋めず、必要なら短く確認する。
- 大きな副作用を伴う操作（破壊的な git 操作、外部システムへの送信など）は事前に確認する。

## コードコメント

- 過剰な説明禁止: 修正の履歴や経緯、以前の仕様との比較はコメントに含めない
- コメントは「Why」が非自明なときだけ書く。コードから読み取れる「What」の説明は書かない。

## 環境

- dotfile は `chezmoi` で管理しており、元のファイルは `~/git/dotfiles` リポジトリ以下にある。`$HOME` 側を直接編集してもソースには反映されないので、必ずリポジトリ側を編集する。
- `mise` がインストールされており、node・python などのランタイムを管理している。
